### Features
#### Hardware
- DirectX Hardware rendering support.
- Transparancy
- Filtering.
#### Software
- Lambert and Phong Shading Model.
- Normal Map Support.
- Different Culling Modes.
- Multithreading.
- AABB Optimization.
